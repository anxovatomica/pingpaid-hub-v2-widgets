🔧 Platform-Specific Instructions

  • WordPress Add a Custom HTML block and paste the iframe code. No plugin needed.
  • Webflow Add an Embed element, paste the iframe code, and set width to 100% with a max-width of 400px.
  • Squarespace Add a Code Block, paste the iframe, and adjust the height based on your layout (600px recommended).
  • Wix Add an HTML Embed element from the Embed section, paste the code, and set dimensions.
  • Ghost/Substack Use the HTML card in the editor and paste the iframe code directly.
  • Notion Unfortunately Notion doesn't support iframes. Link to the calculator page instead.
  • Shopify Add the code to a page template or use an app that supports custom HTML embeds.
  • React/Vue/Angular Use the iframe directly in your JSX/template, or build a custom component using the PingPaid API.
